Hack The Box
邀请码获取

登录Hack The Box

点击Hack The Box进入官网

进入输入邀请码界面

查看源代码

一般按F12即可(查看源代码好处非常多)

进入js界面

response|function|log|console|code|dataType|json|POST|formData|ajax|type|url|success|api|invite|error|data|var|verifyInviteCode|makeInviteCode|how|to|generate|verify

获取邀请代码 即makeInviteCode

在控制获取代码

在控制台输入 makeInviteCode()

如图可以发现编码类型为 rot13

rot13解码

点击进入rot13解码

In order to generate the invite code, make a POST request to /api/invite/generate
为了生成邀请代码，请对 /api/invite/generate 发出POST请求

向网站发送post请求

向 https://www.hackthebox.eu/api/invite/generate 发送post请求

在这我使用 Postman

使用postman

发送post请求

得到如下

发现格式是 encoded 编码的 所以我们还需要转换

解码

点击进入Base64解码

解码结果如下

得到我们的邀请码 ERJGV-MJRON-XPOHP-TENJN-LNFSP

输入邀请码

完成注册
